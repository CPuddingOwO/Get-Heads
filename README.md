
# Get-Heads

---------

Fork 自 [Passion-Never-Dissipate/G0d-of-Heads](https://github.com/Passion-Never-Dissipate/G0d-of-Heads)

---------

一个简单的获取任意数量玩家头颅的插件。

需要 `v2.6.0` 以上的 [MCDReforged](https://github.com/Fallen-Breath/MCDReforged)

##  命令基本说明

`!!head [玩家名称] [数量]` 获取**玩家名称**的头颅*数量*个，当**玩家名称**未指定时，默认自己，当**数量**未指定时，默认为1。

建议获取数量不要超过一组。
